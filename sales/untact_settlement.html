[개발 메모]
- 본 화면의 공급가/부가세는 "매출 기준"이 아닌 "결제수수료 기준"임
- 결제수수료 = 공급가 + 부가세 (부가세 포함 금액)
- 공급가 = 결제수수료 / 1.1 (절사 또는 반올림 정책 필요)
- 부가세 = 결제수수료 - 공급가

- 지급금액 = 결제금액 - 결제수수료 - 환불수수료
- 결제수수료 = 결제금액 × 수수료율 (약 3.7% / 카드사별 상이 가능)

- 승인번호 / 승인일시는 PG 또는 카드사 기준값 사용
- 지급일자는 실제 정산 입금일 기준

- ⚠ 반올림/절사 기준을 반드시 통일 (1~2원 오차 방지)
- ⚠ 환불 발생 시 환불수수료 별도 차감 처리 필요
